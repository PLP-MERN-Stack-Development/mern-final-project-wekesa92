## Quick start

### Backend
```bash
cd backend
cp .env.example .env
# set MONGO_URI and JWT_SECRET in .env
npm install
npm run seed   # optional: seed example data
npm run dev
```

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Open the frontend dev URL and register a user (or use seeded user seed@example.com / password123).

Notes:
- This starter includes improved error handling and a seed script.
- Add further validation, tests, and CI as you like.
